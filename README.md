SLIET Mentorship Platform

A responsive and user-friendly frontend website designed to connect students with mentors for academic and career guidance. The platform aims to bridge the gap between academia and industry by offering 1-on-1 mentorship programs in various technical domains.

Features:

Responsive Design — Works seamlessly across devices (desktop, tablet, and mobile).

Navigation Bar — Easy access to Home, Mentors, About, and Contact sections.

Dynamic Banner Section — Highlights the platform’s purpose and includes a mentor search bar.

Footer with Quick Links & Contact Info — Provides essential links and communication details.

Technologies Used:


HTML — Structure and content

CSS — Styling and responsiveness

JavaScript — Interactivity and dynamic features

Google Material Icons — Icons used for UI enhancement

Google Fonts — Typography styling

How to Run Locally
Clone or download this repository.

https://github.com/yourusername/sliet-mentorship.git

Navigate to the project folder.

cd sliet-mentorship

Open the project in your browser.

Double-click on index.html, or

Serve it locally using VS Code’s Live Server extension.
